# ROS2

## Link
https://docs.ros.org/

## What it is
Middleware for robotics. The de facto standard framework for building robot applications.
ROS2 (Robot Operating System 2) provides tools, libraries, and conventions for writing robot software across platforms.

## When to use
- Creating nodes (basic computation units)
- Understanding topics/services/actions
- Debugging ROS graph
- Building launch files
- Working with parameters

## Key sections
- Concepts (nodes, topics, services, actions, parameters)
- Tutorials (beginner → advanced)
- CLI tools (ros2 run, ros2 topic, ros2 node, ros2 service)
- Client libraries (rclpy for Python, rclcpp for C++)
