# SLAM (Simultaneous Localization and Mapping)

## Link
https://github.com/SteveMacenski/slam_toolbox

## What it is
SLAM allows a robot to build a map of an unknown environment while simultaneously tracking its own position within that map. slam_toolbox is the primary SLAM package for ROS2.

## When to use
- Exploring unknown environments
- Building maps for later navigation
- Real-time localization without prior maps

## Key sections
- slam_toolbox (primary ROS2 SLAM)
- Online vs offline SLAM
- Integration with Nav2
- Map saving and loading

## Related tools
- AMCL (localization with existing map)
- Cartographer (Google's SLAM)
- RTAB-Map (visual SLAM)
