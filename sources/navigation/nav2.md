# Nav2 (Navigation2)

## Link
https://navigation.ros.org/

## What it is
Nav2 is the navigation framework for ROS2. It handles path planning, obstacle avoidance, and autonomous navigation for mobile robots.

## When to use
- Autonomous mobile robot navigation
- Path planning (global + local)
- Obstacle avoidance
- Costmap management
- Behavior trees for navigation logic
