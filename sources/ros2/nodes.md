## What it is
A node is a fundamental building block of ROS2. Each node is responsible for a single, modular purpose (e.g., one node for controlling wheel motors, one for reading a lidar sensor).

## When to use
- Every ROS2 program is built from nodes
- When you need to break robot behavior into modular components
- When different parts of the system need to communicate
