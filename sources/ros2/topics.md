## What it is
Topics are a publish/subscribe communication mechanism. Nodes publish messages to a topic, and other nodes subscribe to receive those messages. This is the primary way data flows in ROS2.

## When to use
- Streaming sensor data (lidar, camera, IMU)
- Sending commands (velocity, joint positions)
- Any continuous data flow between nodes

## Key sections
- Publishers and Subscribers
- Message types (std_msgs, sensor_msgs, geometry_msgs)
- QoS (Quality of Service) profiles
- Topic remapping
