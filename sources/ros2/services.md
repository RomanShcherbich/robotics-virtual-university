## What it is
Services are a request/response communication mechanism. Unlike topics (continuous stream), services are for discrete operations: send a request, get a response.

## When to use
- Triggering one-time actions (e.g., take a photo, save a map)
- Querying state (e.g., get current position)
- Configuration changes (e.g., set parameters)
