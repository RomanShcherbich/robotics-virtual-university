# ROS2 Launch System

## Link
https://docs.ros.org/en/rolling/Tutorials/Intermediate/Launch/Launch-Main.html

## What it is
Launch files start multiple nodes and configure them. Instead of running each node manually, a launch file orchestrates the entire robot system.

## When to use
- Starting a full robot stack (sensors + controllers + navigation)
- Passing parameters to nodes
- Setting up remappings
- Conditional launching

## Key sections
- Python launch files
- XML launch files
- Launch arguments
- Including other launch files
- Event handlers
