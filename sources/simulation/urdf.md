# URDF (Unified Robot Description Format)

## Link
https://docs.ros.org/en/rolling/Tutorials/Intermediate/URDF/URDF-Main.html

## What it is
URDF is an XML format for describing a robot model — its links (bodies), joints, visual appearance, collision geometry, and inertial properties. It's the standard way to describe robots in ROS2.

## When to use
- Defining a robot's physical structure
- Visualization in RViz
- Simulation in Gazebo
- Motion planning with MoveIt
- TF2 frame tree generation

## Key sections
- Links and joints
- Visual, collision, inertial properties
- Xacro (macros for URDF)
- URDF → SDF conversion for Gazebo
