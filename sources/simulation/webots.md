# Webots

## Link
https://cyberbotics.com/doc/guide

## What it is
Webots is an open-source robot simulator. It provides a complete development environment for modeling, programming, and simulating robots. Good alternative to Gazebo with a more user-friendly interface.

## When to use
- Quick prototyping of robot behaviors
- Educational robotics
- When a simpler alternative to Gazebo is needed
- Cross-platform simulation

## Key sections
- Getting Started
- Robot models library
- Programming (Python, C, C++, Java)
- ROS2 integration
- PROTO files (robot descriptions)
