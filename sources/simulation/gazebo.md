# Gazebo

## Link
https://gazebosim.org/docs

## What it is
Gazebo is a 3D robotics simulator. It provides physics simulation, sensor simulation, and environment modeling. The modern version (Gazebo Harmonic/Ionic) replaces the legacy Gazebo Classic.

## When to use
- Testing robot behavior before deploying on real hardware
- Simulating sensors (lidar, camera, IMU, GPS)
- Physics-based testing (collisions, gravity, friction)
- Multi-robot simulation

## Key sections
- Installation guide
