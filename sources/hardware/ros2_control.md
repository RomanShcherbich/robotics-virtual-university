# ros2_control

## Link
https://control.ros.org/

## What it is
ros2_control is the hardware abstraction and control framework for ROS2. It separates hardware interface from control logic, allowing the same controllers to work with different robots.

## When to use
- Interfacing with real hardware (motors, encoders)
- Running controllers (joint position, velocity, effort)
- Simulation ↔ hardware switching
- Mobile base control (diff_drive_controller)

## Key sections
- Architecture (hardware interfaces, controllers, controller manager)
- Available controllers
- Hardware interface plugins
- URDF integration
- Simulation support
