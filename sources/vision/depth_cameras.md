## What it is
Depth cameras (RealSense, ZED, Kinect) produce point clouds and depth images. These are essential for 3D perception, obstacle detection, and manipulation.

## When to use
- 3D obstacle detection
- Object recognition with depth
- Point cloud processing
- SLAM with visual data

## Key tools
- Intel RealSense SDK + ros2 wrapper
- PCL (Point Cloud Library)
- Open3D
- depth_image_proc (ROS2 package)
