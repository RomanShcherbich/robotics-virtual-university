## Link
https://docs.opencv.org/

## What it is
OpenCV (Open Source Computer Vision Library) is the most widely used computer vision library. It provides tools for image processing, object detection, feature matching, camera calibration, and more.

## When to use
- Image processing (filtering, edge detection, color spaces)
- Object detection and tracking
- Camera calibration
- Feature matching
- Integration with ROS2 via cv_bridge
