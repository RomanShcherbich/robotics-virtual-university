# MCAP

## Link
https://mcap.dev/

## What it is
MCAP is a modern file format for recording and replaying robotics data. It's the successor to ROS bag files, designed for performance and cross-platform compatibility.

## When to use
- Recording robot data for later analysis
- Replaying scenarios for debugging
- Data logging for ML training
- Sharing datasets

## Key sections
- MCAP specification
- CLI tools
- ROS2 integration (ros2 bag uses MCAP by default)
- Python API
- Foxglove integration
