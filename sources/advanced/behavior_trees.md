# Behavior Trees

## Link
https://www.behaviortree.dev/

## What it is
Behavior Trees (BT) are a structured way to define complex robot behaviors. Nav2 uses BehaviorTree.CPP for navigation logic. BTs replace finite state machines with a more modular and reusable approach.

## When to use
- Defining complex robot behavior sequences
- Navigation decision-making (Nav2 uses BTs)
- Task planning and execution
- Recovery behaviors

## Key sections
- BT basics (sequence, fallback, decorator, action)
- BehaviorTree.CPP library
- Nav2 behavior trees
- Custom BT nodes
- Groot (BT visual editor)
