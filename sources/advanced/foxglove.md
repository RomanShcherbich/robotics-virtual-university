# Foxglove

## Link
https://foxglove.dev/docs

## What it is
Foxglove is a visualization and debugging tool for robotics. It can visualize ROS2 data (topics, tf, images, maps, point clouds) in a web browser. It replaces/supplements RViz for many use cases.

## When to use
- Visualizing robot data (alternative to RViz)
- Remote debugging (works in a browser)
- Recording and replaying data (MCAP format)
- Multi-panel dashboards

## Key sections
- Getting Started
- Panels (3D, Image, Plot, Map, Log, etc.)
- Data sources (live ROS2, MCAP files)
- Foxglove Bridge (ROS2 → Foxglove)
- Custom extensions
