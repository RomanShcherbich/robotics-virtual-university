# Prompt: Fix Error

## When to use
When you hit an error in code or configuration.

## Template

```
Context: I'm working on {task_description}
Source: {source_file}

Error:
```
{paste error message here}
```

My code:
```python
{paste relevant code}
```

Questions:
1. What does this error mean?
2. What is the root cause?
3. How to fix it?
4. How to prevent it in the future?
```

## Notes
- Always include the full error message
- Include the relevant code
- Reference the source file for context
