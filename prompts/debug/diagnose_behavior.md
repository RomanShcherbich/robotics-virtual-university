# Prompt: Diagnose Unexpected Behavior

## When to use
When code runs without errors but behaves unexpectedly.

## Template

```
Context: I'm working on {task_description}
Source: {source_file}

Expected behavior: {what_should_happen}
Actual behavior: {what_actually_happens}

My code:
```python
{paste relevant code}
```

What I've tried:
- {attempt_1}
- {attempt_2}

Help me:
1. Identify possible causes
2. Suggest debugging steps
3. Explain the fix
```
