# Prompt: Code Review

## When to use
After completing a coding task — review before committing.

## Template

```
Review my code for {task_description}.
Source: {source_file}

```python
{paste your code}
```

Check:
1. Does it follow ROS2 best practices?
2. Are there bugs or edge cases?
3. Is the code clean and readable?
4. What would you improve?
5. Rate: beginner / acceptable / good / excellent
```
