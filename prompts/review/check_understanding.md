# Prompt: Check Understanding

## When to use
Before moving to the next lesson — verify you actually understand.

## Template

```
I just finished learning about {concept} from {source_file}.

Here is my understanding:
{write_your_understanding_in_your_own_words}

Questions:
1. Is my understanding correct?
2. What am I missing?
3. Give me 3 questions to test my knowledge
4. Rate my understanding: beginner / intermediate / solid
```

## Notes
- This is the "no moving forward without understanding" rule
- Be honest about your understanding
- If rated below "solid", go back to the source
