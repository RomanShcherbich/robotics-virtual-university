# Prompt: Compare Concepts

## When to use
When you need to understand the difference between two or more related concepts.

## Template

```
Use {source_file_1} and {source_file_2}

Compare:
- {concept_A} vs {concept_B}
- When to use each?
- What are the trade-offs?
- Show a side-by-side example

Format: table + short explanation
```

## Example

```
Use sources/ros2/topics.md and sources/ros2/services.md

Compare:
- Topics vs Services
- When to use each?
- What are the trade-offs?
- Show a side-by-side example

Format: table + short explanation
```
