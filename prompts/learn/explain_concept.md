# Prompt: Explain Concept

## When to use
When learning a new concept from a source file.

## Template

```
Use {source_file}

Explain:
- What is {concept}?
- How does it work internally?
- Show a minimal Python example
- What are common mistakes?

Level: beginner
Language: keep it simple, use analogies
```

## Example

```
Use sources/ros2/nodes.md

Explain:
- What is a ROS2 node?
- How does it work internally?
- Show a minimal Python example
- What are common mistakes?

Level: beginner
Language: keep it simple, use analogies
```

## Notes
- Always reference a specific source file
- Set the level (beginner / intermediate / advanced)
- Ask for examples, not just theory
