# Prompt: Deep Dive

## When to use
When you've covered basics and need to go deeper into a specific topic.

## Template

```
Use {source_file}

I already understand the basics of {concept}.

Now explain:
- {specific_question_1}
- {specific_question_2}
- {specific_question_3}

Show real-world usage patterns.
Include edge cases and gotchas.
```

## Example

```
Use sources/ros2/tf2.md

I already understand the basics of TF2 transforms.

Now explain:
- How does the transform tree handle time?
- What happens with missing transforms?
- How to debug transform issues?

Show real-world usage patterns.
Include edge cases and gotchas.
```
