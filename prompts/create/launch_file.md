# Prompt: Create Launch File

## When to use
When you need to generate a ROS2 launch file.

## Template

```
Use sources/ros2/launch.md

Create a ROS2 Python launch file that:
- Launches {node_1} with parameters: {params}
- Launches {node_2} with parameters: {params}
- {additional_requirements}

Include:
- Proper imports
- generate_launch_description() function
- Launch arguments where appropriate
- Comments explaining the setup

ROS2 distro: Jazzy
```
